# Campus Placement Analysis – Power BI Dashboard

## Project Overview

The **Campus Placement Analysis Dashboard** is a data visualization project developed using **Microsoft Power BI** to analyze factors that influence student placements. The dashboard provides insights into placement trends, salary distribution, work experience impact, and specialization performance.

The objective of this project is to help colleges, placement cells, and students understand the patterns in placement data and make data-driven decisions.

---

## Dataset

The dataset used in this project is taken from Kaggle.

Dataset Link:
[https://www.kaggle.com/datasets/benroshan/factors-affecting-campus-placement](https://www.kaggle.com/datasets/benroshan/factors-affecting-campus-placement)

The dataset contains information about students' academic background, specialization, work experience, and placement outcomes.

---

## Tools and Technologies

* **Power BI Desktop** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculated measures and metrics
* **Excel / CSV Dataset** – Data source

---

## Dataset Features

The dataset includes the following attributes:

* **Gender** – Student gender
* **SSC Percentage** – Secondary school percentage
* **HSC Percentage** – Higher secondary percentage
* **Degree Percentage** – Undergraduate degree score
* **Degree Type** – Science, Commerce, etc.
* **Work Experience** – Whether the student has prior work experience
* **Specialisation** – Marketing & Finance or Marketing & HR
* **MBA Percentage** – Postgraduate score
* **Placement Status** – Placed or Not Placed
* **Salary** – Salary offered to placed students

---

## Dashboard Features

The Power BI dashboard includes multiple visualizations to analyze placement trends.

### KPI Cards

* Total Students
* Total Placed Students
* Placement Rate
* Average Salary

### Visualizations

* **Bar Chart:** Placement status by gender
* **Pie Chart:** Distribution of specialization
* **Column Chart:** Salary comparison by specialization
* **Line Chart:** Placement trend based on academic scores
* **Table:** Detailed student placement information

### Filters (Slicers)

The dashboard includes interactive filters for:

* Gender
* Degree Type
* Specialisation
* Work Experience

These filters allow users to dynamically explore placement trends.

---

## Key Insights

The dashboard helps identify important placement patterns such as:

* Students with **work experience tend to have higher placement rates**.
* **Marketing & Finance specialization** often shows higher salary packages.
* Higher **academic percentages correlate with better placement chances**.

These insights can assist placement cells in planning training programs and guiding students toward better career opportunities.

---

## How to Use the Dashboard

1. Open the **Power BI (.pbix) file**.
2. Load the dataset if required.
3. Use the **interactive slicers** to filter data.
4. Analyze placement trends using charts and KPIs.
